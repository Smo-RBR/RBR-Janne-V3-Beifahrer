**Modifier**

**Narrows** \= The road gets narrower.  
**Wideout** \= Usually used with a corner where there’s room on the outside to extend the driving line onto grass, lay-by or something similar. E.g. *Square Left, **Wideout**, 80\.*  
**Tightens** \= The corner tightens towards its end. I personally use the Tightens-note when the corner tightens only so much that it doesn’t become a tighter note on the scale yet.   
**Double Tightens** \= The corner tightens a lot towards its end. I rarely use this note, but instead the more specific notes introduced on the Tightens-tab.   
**Cut** \= Cut the corner from the inside for a half a car’s width, so you can take the corner faster.  
**Don’t (or Don’t Cut)** \= There’s an obstacle of some sort that prevents you from cutting the corner. Drive on the road.  
**Long** \= The corner is longer than a corner without a length definition, so you’ll need to drive it differently than your “normal” corner of the same scale.  
**Maybe** \= Multiple different uses for this one. I don’t use this note myself. I’ve heard stories of some older Finnish drivers using the Maybe-note when you could go a bit faster into the corner than the note specifies. E.g. if you weren’t sure if the corner was a Flat Right or an Easy Right, you’d mark it as Easy Right Maybe. Although using it in this manner is a bit redundant as you can test and edit your pacenotes as much as you want.
